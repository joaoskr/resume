Basically this is my personal website.
I turned it into opensource to help people who are looking for information about applications using JS stack.

## Installation

Use the package manager [yarn](https://yarnpkg.com/) to install.

```bash
yarn install
```

## Usage

Use the command **yarn start** to run it locally
```bash
yarn start
```

Once you finished the project, you must run **yarn build** to build and be ready to deploy
```bash
yarn build
```
